# FastAPI_Learning
Код писался при помощи PyCharm на версии Python 3.12
Перед запуском кода необходимо иметь основные библиотеки FastAPI (для их установки можно использовать команду pip install 'fastapi[standart]') 
Проект запускается командой python main.py, после чего через браузер открывается инструмент взаимодействия с API - Swagger UI (документация AP)
Добавьте /docs после 127.0.0.1:8000, чтобы открылась документация API (127.0.0.1:8000/docs)
Основные запросы:
get/tasks - получает список всех задач
post/task - Создает новую задачу 
get/tasks/{task_id} - получает конкретную задачу ()
put/tasks/{task_id} - обновляет задачу по id
delete/task/{task_id} - удаляет задачу по id
